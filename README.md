# ShopEasy E-Commerce Store

A simple E-Commerce Store built using HTML, CSS, JavaScript, Node.js, Express.js and MongoDB.

## Features

- Product Listing
- Product Details Page
- Shopping Cart
- Product Search
- User Registration
- User Login
- Checkout Page
- Order Processing
- REST API
- MongoDB Database Integration

## Technologies Used

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB
- Mongoose

## Installation

1. Clone repository

2. Install dependencies

npm install

3. Run server

node server.js

4. Open browser

http://localhost:5000/api/products
